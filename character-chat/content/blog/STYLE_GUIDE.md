# Agentwood Master Blog Strategy & Style Guide

## 1. Brand Voice: "Calm. Intelligent. Certain."
- **Tone:** Founder / Operator mindset.
- **Rules:** No hype ("revolutionary"). No fluff ("In today's world"). No AI slop.
- **POV:** "We" (Team/Founder).
- **Core Themes:** Memory, Continuity, relationships, Agents vs Chatbots.

## 2. Founder Voice vs Neutral Brand Voice
- **Founder:** Opinionated, personal conviction ("We believe..."), strategic insight.
- **Neutral:** Objective, educational, platform-level authority.

## 3. Anti-AI Slop Checklist
- [ ] Remove generic openers ("In the fast-paced world...").
- [ ] Remove generic transitions ("Furthermore", "Additionally").
- [ ] Remove empty hype.
- [ ] Add human signals: Slight opinion, selective emphasis, varied sentence length.

## 4. 90-Day Plan (Month 1: Foundational)
- **Week 1:** AI characters with memory, Why chatbots forget, Stateless vs persistent.
- **Week 2:** Long-term memory systems, Context vs prompts.
- **Week 3:** Character bots vs AI agents.
- **Week 4:** Memory as infrastructure.

## 5. Daily Routine
- **Post A:** Evergreen/Educational (Agentwood feature).
- **Post B:** Industry/News (Analysis).
